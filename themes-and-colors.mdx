# Mintlify Themes and Colors

The following built-in Mintlify themes are available for `docs.json`.

## Theme list

- `mint`
- `maple`
- `palm`
- `willow`
- `linden`
- `almond`
- `aspen`
- `sequoia`
- `luma`

## Example configuration

```json
{
  "theme": "mint",
  "colors": {
    "primary": "#0D9373",
    "light": "#6ADBB5",
    "dark": "#0B7A60"
  }
}
```

Switch the `theme` value and color values as needed for your branding.
